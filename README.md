\# Stubble Burning Air Quality ML



\## Overview

This project investigates the impact of agricultural stubble burning in Punjab and Haryana on air quality in India's National Capital Region (NCR).



\## Data Sources

\- MODIS MOD14A1

\- MODIS MCD64A1

\- Sentinel-5P TROPOMI

\- ERA5 Reanalysis

\- NOAA HYSPLIT



\## Methods

\- Fire Activity Analysis

\- Correlation Analysis

\- Anomaly Detection

\- Linear Regression

\- Random Forest

\- XGBoost

\- Back-Trajectory Analysis



\## Key Findings

\- Burned area showed the strongest correlation with pollution anomalies.

\- Random Forest achieved the best predictive performance.

\- Aerosol Index was most strongly linked to stubble burning activity.



\## Technologies

Python, Google Earth Engine, Pandas, NumPy, Matplotlib, Scikit-Learn, XGBoost
